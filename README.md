# Mad Libs Game 📝

Welcome to the **Mad Libs Game**! This is a fun, interactive Python command-line application that generates creative stories based on user input.

## 🚀 Features

- **3 Unique Templates:** Choose between a Hospital visit, a Camping trip, or an Enchanted Castle adventure.
- **Dynamic Input:** The program asks for specific types of words (nouns, adjectives, verbs, etc.) to build a unique story every time.
- **Randomized Endings:** Every story ends with a fun, randomized concluding remark.
- **Modern Python:** Uses `f-strings` for clean and readable string formatting.
